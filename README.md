## 🗂 Projects

### 🔗 [Library System Management](https://github.com/HUB37T/Projek-Akhir)
Library System Management is a Java-based desktop application built with a GUI interface to simplify library operations for both users and administrators.
Users can browse available books, borrow books, and manage their reading activities through an interactive and user-friendly system.
On the administrative side, the application supports full CRUD functionality, allowing admins to add, edit, update, and remove books efficiently.
This project focuses on combining object-oriented programming concepts with graphical interface development to create a practical library management experience.

### 🔗 [Senandika](https://github.com/JavaneseIvankov/senandika)
Senandika is an AI-powered web platform where users can share their thoughts and feelings, and receive thoughtful responses through conversational AI built with Next.js and React.
The platform introduces gamification, allowing users to earn badges and rewards as they interact more frequently and build consistent engagement.
This project blends emotional support with playful progression, creating a safe and motivating space for self-expression.

### 🔗 [Pangantara](https://github.com/fahrypratama17/pangantara)
Pangantara is a web platform built with Next.js that acts as a bridge between SPPG in Malang and suppliers providing essential cooking ingredients.
The system is designed to streamline communication, manage supply needs, and ensure that resources flow efficiently from suppliers to SPPG.
By connecting demand and supply in a structured way, Pangantara helps support operational continuity and reduces friction in the procurement process.

### 🔗 [FilkomEvent](https://github.com/fahrypratama17/FilkomEvent)
FilkomEvent is a web-based platform that centralizes event information within the Faculty of Computer Science, Universitas Brawijaya.
It allows students to discover events, view details, register online, and track their participation status in one place.
On the admin side, it supports event management, participant handling, and streamlined publication, making the overall process more organized and efficient.

### 🔗 [Pendar](https://github.com/fahrypratama17/Pendar)
Pendar is a cross-platform mobile mental health application designed for university students, integrating an XGBoost machine learning model to predict burnout levels in real time based on users' psychological metrics.
The app provides a Mind Check feature for burnout risk analysis, guided 4-7-8 breathing interventions, and a digital journal with auto-save to support ongoing self-reflection.
It also includes task and schedule management to help students stay organized throughout their recovery journey.
By combining predictive analytics with practical mental health tools, Pendar offers a holistic approach to student wellbeing.
